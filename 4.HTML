인터넷과 웹은 차이가 있을까요?
인터넷이 도로라면, 웹은 도로위를 달리는 자동차 한대!
인터넷 안에 웹, 서비스, 이메일, FTP 등의 수많은 것이 구성되어있습니다.

웹의 시작점은 이곳이다. 웹에 메소포타미아에 왔다. http://info.cern.ch/
1990년부터 웹이 등장하며 인터넷이 대중화되기 시작한다. 쓸만하다!

인터넷이 동작하기 위해서는 적어도 2대의 컴퓨터가 있어야 합니다. 그리고 두개의 프로그램, 하나는 웹브라우저 프로그램 나머지는 웹서버 프로그램

서로 정보를 주고받죠. web browser request to web server, web server response to web browse
web browser = client, web server = server
우리는 웹 브라우저에 익숙하지만, 웹 서버 에는 약간 생소할 수도 있어요.
둘다 다운해서 쓸수 있는 프로그램이니, 걱정마세요. 그냥 익숙해지면 됩니다.

웹서버를 직접 깔것이냐, 웹 호스팅 대행 업체에 맡기느냐?
보통 웹서버는 어려우니, 처음에는 웹 호스팅 맡기는 걸 추천합니다!

우리가 직접 웹서버를 운영하는 것은 어려워요. 항상 컴퓨터를 연결해야하거든요. 인터넷이 연결된 컴퓨터 = 호스트(host)
웹호스팅 업체 검색해보면 엄청 많아요.
